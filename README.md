# Paris 2024 Experience

This project is a responsive and interactive web application built around the Paris 2024 Olympic Games. It offers a structured experience for exploring sports, athletes, teams, venues, competitions, medals, and other related content in a dynamic and visually appealing interface.

The website was developed as a front-end project focused on usability, organization, and presentation. It combines multiple pages and sections to create a digital experience inspired by the official Olympic theme, while also including interactive features such as a countdown, a quiz, and a store section.

Some content may require access through the university VPN and a running local server in order to display correctly, depending on the data sources and environment configuration.

## Technologies Used

This project was built using a combination of core web technologies and UI libraries:

- HTML5 for the page structure and content organization
- CSS3 for styling, layout, and visual design
- JavaScript for interactivity and dynamic behavior
- Bootstrap 5 for responsive layout components and navigation
- Font Awesome for icons and visual enhancement
- Leaflet for map-based content and location visualization

## Main Features

- Navigation through different Olympic-related sections
- Detailed pages for sports, athletes, teams, venues, and competitions
- Interactive quiz and countdown sections
- Responsive design for improved browsing across devices
- Visual elements and styling inspired by the Paris 2024 theme

## How to Run

Open the main entry file (index.html) in a browser, or run the project using a local development server such as Live Server in Visual Studio Code.
